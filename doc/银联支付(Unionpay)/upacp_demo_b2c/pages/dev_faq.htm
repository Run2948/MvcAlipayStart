常见.net问题：

System.Security.Cryptography.CryptographicException: 指定了无效的提供程序类型

System.Security.Cryptography.CryptographicException: 拒绝访问/Access is denied

System.Security.Cryptography.CryptographicException: The system cannot find the file specified. （系统找不到指定的文件）

System.Security.Cryptography.CryptographicException: 出现了内部错误/An internal error occurred

验签失败：
请检查一下配置文件的sdk.validateCert.dir配置的文件夹下有对应的验签cer文件，如切换正式环境请注意放一下assets文件夹下的正式环境的cer。
